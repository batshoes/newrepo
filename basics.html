HTML TAGS

There are a number of tags used in the Hyper Text Markup Language. Most of them have meaning and are specifically designed to have influence over the text that they contain. For the most part, tags are opened and closed and encased in angle brackets. You may know them as 'less than - <' or 'greater than - >'.

<!DOCTYPE html>

This tells the document you are about to write that it is going to be written in HTML.

<head>

The head tag is used to hold links to external webpages, sources, javascripts an stylesheets. It is essential to every HTML document.

<title>

<body>

The body tag is used to contain everything the user sees on a webpage. It is essential to every HTML document.

<h1-6> - Heading

h1 is used for main headings.
There are other headings all the way down to h6, the smallest of headings and are "wrapped" around text as such
<h3> this is a sub heading </h3>

<p>

The p tag is used to contain medium to large bodies or "paragraphs" of text.

<a>

An "a" tag actually stands for an "anchor" tag. An anchor tag is a link to an external web page, another part of the current web page, or another form of the same webpage. They come in the form of
<a href='path_to_website'>'words_or_image_to_be_clicked'</a>

<div>

A div tag is basically a wrapper, used to confine text/images/content into a specific area. <div>content_of_div</div>

<span>

A span tag is used to wrap a particular length of words so that they can be manipulated using CSS. <span>This is now italicised from the span </span>

<img>

An img tag is an Image. Unlike many other tags, this particular tag does not need to be closed. It can acquire an image from the internet, or from a local folder, relative to the page it is being called on.
<img src="URL_or_Path_to_image.jpg" alt="This_is_a_backup_image.jpg">

<button>

Button tags are used to create, you guessed it, buttons. They have specific CSS attributes already atatched to them.

<form>

Defines a HTML form that a user can input information to.

<ul>

This is an unordered list tag. Used to surround 'li' tags. Defines a list.

<li>

Stands for List Item. Each of these is specifically designed to hold an element of the list.
